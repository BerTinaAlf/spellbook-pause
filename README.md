# Spellbook Pause

Replaces Foundry VTT's default pause overlay with a spellbook GIF and built-in randomized adventure text.

## GIF

Put your GIF at:

```text
D:\fvtt\FoundryVTT\Data\modules\spellbook-pause\assets\spellbook-pause.gif
```

## Settings

- `暂停动画位置`: center of the screen or lower quarter of the screen.

The random pause text is built into the module and is not configurable from Foundry settings.
The spellbook image path is fixed at `modules/spellbook-pause/assets/spellbook-pause.gif`.
